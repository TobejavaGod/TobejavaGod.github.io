EL表达式

概念：Expression Language表达式语言

作用：替换和简化JSP页面中Java代码的书写

语法：${表达式}

在JSP中会默认解析EL表达式，如果不想让jsp解析EL表达式，可以在page中设置isELIgnored的值为true，但是这样的话整个页面的EL表达式都会忽略，如果想要忽略某一个EL表达式的话可以在那一条EL表达式前面加上反斜线  -->  "\"

使用

1.运算
    *运算符
        1.算术运算符：/(div)  %(mod)
        2.比较运算符
        3.逻辑运算符:&&(and)  ||(or)  !(not)
        4.空运算符：
            empty:用于判断字符串、集合、数组对象是否为null并且长度是否为0
                ${empty list}  -->   判断list这个键对应的值是否为空或长度是否为0
                ${not empty list}  -->  判断是否不为null并且长度大于0
2.获取值 --> 注意：获取对象时获取的是对象的属性值，属性就是Javebean中的set或get方法去掉set后将首字母变小写的结果，所以可以直接使用'.'，如果没有对应得get方法得话会报错
    1.EL表达是只能从域对象中获取值
    2.语法
        1.${域名称.键名}：从指定域中获取值
            *域名称
                1.pageScope   --> pageContext
                2.requestScope --> Request
                3.SessionScope --> session
                4.applicationScope --> application(ServletContext)
            *例如：在request域中设置了name为张三
            *获取：${RequestScope.name} --> 如果没用获取到的话返回空字符串
        2.${键名}：表示依次从最小的域中查找，知道找到为止
        3.获取对象：${域名称.对象名.属性名} --------> 本质上会调用对象的getter方法
            *逻辑视图（很常用）
                比如要显示一个格式化的日期，可以在javabean中添加一个getDateFormat函数，在方法中格式化并返回，在jsp中可以直接使用对象.dateFormat获取。
        4.获取List集合中的元素： ${域名称.键名[索引]}
            ${list[0]}如果填写的下标越界不会抛出异常，而是什么都不显示。
        5.获取map集合元素：${域名称.键名.key名称}
                          ${域名称.键名["key名称"]}
    3.隐式对象：
        *EL中有11个隐式对象
        *pageContext：
            获取其他8个隐式对象 --> 最常见的就是获取request对象  --> request可以获取虚拟目录(contentPath) --> 在jsp中动态的获取虚拟目录  --> ${pageContext.request.contentPath}
